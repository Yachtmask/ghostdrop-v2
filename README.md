# GhostDrop 💀

GhostDrop is a decentralized dead-man's switch vault built on Shelby.

Users encrypt sensitive files locally, assign trusted recipients, and set a check-in timer. If the user stops checking in, the encrypted vault is automatically released.

## Core Features

• AES-256-GCM file encryption  
• Zero-knowledge storage on Shelby  
• Recipient-specific key packages  
• Dead-man switch vault release  
• Watchdog daemon monitoring

GhostDrop ensures that no server, company, or third party ever has access to your files or encryption keys.
